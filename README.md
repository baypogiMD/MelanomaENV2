Welcome to MelanomaENV2

The model classifies a given dermoscopy image whether it is a melanoma or not.
Basically, this is a binary classification model.
\n
Performance Metrics:
Accuracy: 80%
Recall: 87%
Precision: ~71%
F1 Score: ~78.08%
